Settings:

log
Wither to display logging.
Defaults to false
subMenuTitle
Sector of a submenu's title in the html.
Defaults to 'span:first'
subMenu
Selector of a submenu in html.
Defaults to 'ul:first'
rootTitle
Text to use for the top level or 'root' menu title.
Defaults to 'Menu'
current
Selector of the submenu to display from the start. Useful for navigation menus.
Defaults to null
up
Selector of an "up" button to hide when the menu is fully at the top.
Defaults to null
down
Selector of a "down" button to hide when the menu is fully scrolled.
Defaults to null
holderSelector
Container for the marked up menu. If null is passed in a div with the class of gfu-menu-holder is created before the "real" menu markup. If a container is given the menu will appear there and given the class gfu-menu-holder.
Defaults to null
itemClick
Function to run when a normal list item is clicked.
Defaults to null
afterSubMenu
Function to run after a submenu is loaded.
Defaults to null

realMenu: null, nav : null, view : null, menu : null
